# Auth Security


## Overview

Auth Security Samples with [AppID](https://www.ibm.com/jp-ja/products/app-id).


## Environment values
- Set following two values for each sample applications:

  - `APPID_REGION`
    - Region
    - Mandatory

  - `APPID_TENANTID`
    - Tenant ID
    - Mandatory

  - `APPID_APIKEY`
    - API Key
    - Mandatory

  - `APPID_CLIENTID`
    - Client ID
    - Mandatory

  - `APPID_SECRET`
    - Client Secret
    - Mandatory

  - `APPID_REDIRECT_URI`
    - Callback URI
    - Mandatory
